Togglers

Description:

Toggler is a widget that has two components -- a button, and a set of contents. When you click the button, the button and content states are toggled.

Examples:

Simple (Nested) -- Single button / single content element

Simplest scenario -- one button and one content element, both wrapped in a uranium-set element. Note that the data-ur-state attribute is optional.

Flat -- Single button / single content element

Another common use case -- one button and one content element. This version doesn't require them to be in a wrapper, but they both must have matching data-ur-id values. The accordions are constructed based on these unique id groups.

Dialog popup -- something besides an accordion

Make a popup. This (and all the examples above) -- use lazy styling. The styling associated with the different actions is also associated with the default styles for that content. In these cases, we aren't really using the 'disabled' data-ur-state --- we just style on the 'enabled' state.

Make multiple content elements toggle

The toggler is designed to toggle any 'content' components that are bound to its group. You can use this to toggle multiple things at once. You can also use the data-ur-state to set the initial state of each of the contents.
