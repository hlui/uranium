Select Buttons

Description

This widget allows you to bind increment/decrement buttons to trigger the appropriate changes on a <select> element.

Components

The widget should be wrapped in a div with the attribute data-ur-tabs-component="select-buttons".

We give the select tag (which produces the drop-down menu) an attribute of data-ur-select-buttons-component="select". Within the select tag, we have a series of option tags. Each of t******

The button aspect of this widget comes in next. There should be two buttons - one to increment the selection and one to decrement the selection. These are given the (logical) tags of:

  • data-ur-select-buttons-component="increment" - for the increment
  • data-ur-select-buttons-component="decrement" - for the decrement
