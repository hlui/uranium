Select Buttons

Description

This widget allows you to bind increment/decrement buttons to trigger the appropriate changes on a <select> element.

Components

Required

  • data-ur-set="select-buttons" - add this attribute to a div that wraps the whole widget
  • data-ur-select-buttons-component="select" - this attribute should be applied to the select tag, which contains the option tags and produces the drop-down menu
  • data-ur-select-buttons-component="increment" - for the increment button
  • data-ur-select-buttons-component="decrement" - for the decrement button

Optional

  • data-ur-state="disabled" - add this to the decrement button

Demonstration

Widget

HTML

[+] [-]

<div data-ur-set='select-buttons'>
  <select data-ur-select-buttons-component='select'>
    <option selected='true' value='-1'> -None- </option>
    <option value='SKU-XS'> XS </option>
    <option value='SKU-S'> S </option>
    <option value='SKU-M'> M </option>
    <option value='SKU-L'> L </option>
  </select>
  <span data-ur-select-buttons-component='increment'> [+] </span>
  <span data-ur-select-buttons-component='decrement' data-ur-state='disabled'> [-] </span>
</div>
