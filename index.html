What is Uranium?

Uranium is a set of "widgets" - snippets of code that allow you to do neat things on your website. It's written in javascript, but requires no knowledge of the language. It's incredibly easy to use.

HOW TO USE:

  1. Give elements pre-defined attributes and values
  2. Include the Uranium library
  3. .. That's it

Why?

It's Lightweight

The whole Uranium library is 8.4 KB zipped! This includes all the widgets and xui.

Xui is a featherweight mobile js library that was used to develop Uranium. Uranium is bundled with xui to make just one javascript file. This makes it perfect for mobile devices, where minimizing the amount of data downloaded is key. If you want it even lighter, you can make a custom build yourself.

It's Easy to Use

Uranium requires zero programming on your part. That's right, you don't need to touch a single line of javascript. It uses the 'declarative' javascript style, a model that looks for how html elements are formatted. Then, it constructs all the necessary js magic to make those elements come to life.
