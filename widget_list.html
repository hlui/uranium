Widgets

What is a Widget?

Uranium is a group of widgets - but what is a widget, exactly? It's not so easy to describe. In Uranium, a widget is a behavior that you're assigning to elements on a page.

Widgets Included in Uranium

  • Toggler
  • Tabs
  • Carousel
  • Select List
  • Select Buttons
  • Zoom Preview
  • Font Resizer
  • Maps
  • Geocode
