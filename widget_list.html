Widgets

What is a Widget?

Uranium is a group of widgets - but what is a widget, exactly? It's not so easy to describe. In Uranium, a widget is a behavior that you're assigning to elements on a page.

Again, in Uranium, a widget is a group of attributes you can give to some elements on a page which allows the elements to perform a function.

Widgets Included in Uranium

  • Tabs
  • Carousel
  • Select List
  • Select Buttons
  • Zoom Preview
  • Font Resizer
