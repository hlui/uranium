Font Resizer

Description

This widget allows us to change the font size of a specified page element.

To use this widget, we need to use a couple of attributes. We can use the set method (i.e. grouping all of the widget components in a single div with the attribute data-ur-set) or the id method (i.e. giving each component of the widget a data-ur-id with an identical value, which doesn't require them to be grouped inside a single wrapper).

Components

Required

data-ur-font-resizer-component=

  • "increase" - this is used to mark the increase button (e.g. [+]).
  • "label" - this indicates the text size.
  • "decrease" - this is used to mark the decrease button (e.g. [-]).
  • "content" - this marks the content.

Optional

It's also possible to override the default maximum, minimum, and initial sizes, as well as how much the size will change on each increment, by adding the following extra attributes to the content element:

  • data-ur-font-resizer-min - defaults to "100"
  • data-ur-font-resizer-max - defaults to "200"
  • data-ur-font-resizer-size - defaults to min
  • data-ur-font-resizer-delta - defaults to "20"

Demonstration

Widget

HTML

[+] [-]

To be or not to be, that is the question. Whether 'tis nobler in the mind to endure the slings and arrows of outrageous fortune etc etc.

<div data-ur-set="font-resizer">
  <span data-ur-font-resizer-component="increase">[+]</span>
  <span data-ur-font-resizer-component="label"></span>
  <span data-ur-font-resizer-component="decrease">[-]</span>
  <p data-ur-font-resizer-component="content">To be or not to be, that is the question. Whether 'tis nobler in the mind to endure the slings and arrows of outrageous fortune etc etc.</p>
</div>
