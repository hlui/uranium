Note: If you're testing this locally (off of a file in your browser) -- then geolocation will not work. Put it in your dropbox to play w these features

Components

data-ur-set="map"

data-ur-map-component

  • "canvas" - sets the map canvas
  • "address" - defines the address for a map point
  • "description" - defines a description for the address
