Tabs

Description:

Tabs are kind of like togglers, but allow a state to be defined.

Examples:

Simple

Firstly, we need to define the buttons for the widget. They have the attribute data-ur-tabs-component with the value "button". So far so good. Accordingly, the content should have the same attribute but with the value of "content".

Now we need to make sure the buttons match the content. We give each button a data-ur-tab-id, and the content we wish to activate is given the same id.

Finally, we need to set the states of the tabs. One tab we will want to show. The others we want to remain hidden. Putting that in terms from the toggler, we want one tab to be enabled and the others to be disabled. All we need to do is specify the button and content we want enabled with the attribute data-ur-state="enabled".

Here's an example of some HTML for a set of tabs:

<div data-ur-set='tabs'>
  <span data-ur-tabs-component='button' data-ur-tab-id='first' data-ur-state='enabled'>
  First </span>
  <span data-ur-tabs-component='button' data-ur-tab-id='second'>
  Second </span>
  <span data-ur-tabs-component='button' data-ur-tab-id='third'>
  Third </span>
  <div data-ur-tabs-component='content' data-ur-tab-id='first' data-ur-state='enabled'>
    <img/>
  </div>
  <div data-ur-tabs-component='content' data-ur-tab-id='second'>
    <img/>
  </div>
  <div data-ur-tabs-component='content' data-ur-tab-id='third'>
    <img/>
  </div>
</div
