Tabs

Description

Tabs are kind of like togglers, but allow a state to be defined. This means that (unlike togglers) only one tab is active at a time. When you enable the state of one tab, another tab is disabled.

Components

Here's a list of attributes to add to HTML to make the widget work

  • data-ur-set="tabs" - add this attribute to a div that wraps the whole widget
  • data-ur-tabs-component="button" - add this attribute to the button (i.e. the thing to be clicked)
  • data-ur-tabs-component="content" - add this attribute to all the contents
  • data-ur-tab-id="NAME" - give matching IDs to button and the content to be toggled
  • data-ur-state="enabled" - assign this attribute to the button and content that will be activated when the page loads

    <img ONE />
  </div>
  <div data-ur-tabs-component='content' data-ur-tab-id='second'>
    <img TWO />
  </div>
  <div data-ur-tabs-component='content' data-ur-tab-id='third'>
    <img THREE />
  </div>
</div
