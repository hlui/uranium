Select List

Description:

This widget allows you to bind a non-select list to trigger changes on a <select> element.

Examples:

Simple example.

A <ul> is the trigger list in this case.

  • -None-
  • XS
  • S
  • M
  • L

Subset example

A <div> is the trigger list in this case. Notice how some options were omitted from the trigger list. This is a valid way to construct the trigger list.
