Select List

Description

This widget allows you to bind a non-select list to trigger changes on a <select> element.

Components

Here's a list of attributes to add to HTML to make the widget work

  • data-ur-set="select-list" - add this attribute to a div that wraps the whole widget
  • data-ur-select-list-component="select" - this attribute should be applied to the select tag, which contains the option tags and produces the drop-down menu
  • data-ur-select-list-component="content" - this should be applied to the list of items that will appear outside the drop-down; each should have a value corresponding to a value on one of the option tags in the select list.

Demonstration

A <ul> is the trigger list in this case.

Widget

HTML

  • -None-
  • XS
  • S
  • M
  • L

<div data-ur-set='select-list'>
  <select data-ur-select-list-component='select'>
    <option selected='true' value='-1'> -None- </option>
    <option value='SKU-XS'> XS </option>
    <option value='SKU-S'> S </option>
    <option value='SKU-M'> M </option>
    <option value='SKU-L'> L </option>
  </select>
  <ul data-ur-select-list-component='content'>
    <li value='-1'> -None- </li>
    <li value='SKU-XS'> XS </li>
    <li value='SKU-S'> S </li>
    <li value='SKU-M'> M </li>
    <li value='SKU-L'> L </li>
  </ul>
</div>
