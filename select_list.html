Select List

Description

This widget allows you to bind a non-select list to trigger changes on a <select> element.

Components

As with all the Uranium widgets, we first need to wrap the widget in a div with the attribute data-ur-set="select-list".

Then we have two components to consider. One is the <select> tag, which produces the drop-down list of options. The select tag has an attribute of data-ur-select-list-component="select".
Each <option> tag within the select tag should have a unique value.

The second component is the list of items (it could be a <ul>, for example). The list has an attribute of data-ur-select-list-component="content". The items in the list should have a value corresponding to the value they will select in the drop-down menu. E.g. if we want the "medium" option in the drop-down selected when we click on the "medium" list option, give them both the attribute value="medium".

Demonstration

Simple example.

A <ul> is the trigger list in this case.

Widget

HTML

  • -None-
  • XS
  • S
  • M
  • L

<div data-ur-set='select-list'>
  <select data-ur-select-list-component='select'>
    <option selected='true' value='-1'> -None- </option>
    <option value='SKU-XS'> XS </option>
    <option value='SKU-S'> S </option>
    <option value='SKU-M'> M </option>
    <option value='SKU-L'> L </option>
  </select>
  <ul data-ur-select-list-component='content'>
    <li value='-1'> -None- </li>
    <li value='SKU-XS'> XS </li>
    <li value='SKU-S'> S </li>
    <li value='SKU-M'> M </li>
    <li value='SKU-L'> L </li>
  </ul>
</div>
