Lazy styling vs Proper styling

Popup -- with Lazy Styling

This (and the next) toggler use a button to make an image appear.
However, this popup uses "lazy styling". What is "lazy styling", exactly? It's when we aren't really styling the 'disabled' data-ur-state --- we just style on the 'enabled' state.
Popup !

The pop-up appears way up the page. Not good.

Popup -- with Proper Styling

This time, we'll use proper styling for the popup. We add the data-ur-state attributes so that we can compartmentalize styles.

.popup_content[data-ur-state='disabled'] {
  display: none;
  opacity: 0.5;
}
.popup_content[data-ur-state='enabled'] {
  display: inline-block;
  opacity: 1.0;
}

Popup !
