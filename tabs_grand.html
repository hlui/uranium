Tabs - Basic

Description

Tabs are kind of like togglers, but allow a state to be defined. This means that (unlike togglers) only one tab is active at a time. When you enable the state of one tab, another tab is disabled.

Attributes

  • data-ur-set="tabs" - add this attribute to a div that wraps the whole widget
  • data-ur-tabs-component="button" - add this attribute to the button div
  • data-ur-tabs-component="content" - add this attribute to the content
  • data-ur-tab-id="NAME" - this attribute needs to link the button and the content (i.e. the button and content should have the same id)

Demonstration

Widget

HTML

First Second Third

<div data-ur-set='tabs'>
  <span data-ur-tabs-component='button' data-ur-tab-id='first' data-ur-state='enabled'>

  </span>
  <span data-ur-tabs-component='button' data-ur-tab-id='third'>
    Third
  </span>
  <div data-ur-tabs-component='content' data-ur-tab-id='first' data-ur-state='enabled'>
    <img ONE />
  </div>
  <div data-ur-tabs-component='content' data-ur-tab-id='second'>
    <img TWO />
  </div>
  <div data-ur-tabs-component='content' data-ur-tab-id='third'>
    <img THREE />
  </div>
</div

Tabs - Advanced

Description

Tabs are kind of like togglers, but allow a state to be defined. This means that (unlike togglers) only one tab is active at a time. When you enable the state of one tab, another tab is disabled.

Attributes

data-ur-set=

"tabs"

data-ur-tabs-component=

  • "button"
    • add this attribute to the button (i.e. the thing to be clicked)
    • multiple?: true
    • state: enabled / disabled(default)
    • attributes
      • required: data-ur-tab-id is required and must match attribute on content element
  • "content"
    • add this attribute to each content item
    • multiple?: true
    • state: enabled / disabled(default)
    • attributes
      • required: data-ur-tab-id is required and must match attribute on button element

Instances

Each instance is available by set id under Ur.Widgets["tabs"]. Each instance exposes elements property -- raw components of the widget
