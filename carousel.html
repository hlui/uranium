Carousels

Description

The carousel is a widget that allows horizontal scrolling (with touch or buttons) between a set of items.

The CSS we need to use here is pretty picky, so make sure to check out the CSS tab below.

The compatibility of this with FireFox is ... well, not really there. Check out our compatibility tables to see which devices and browsers will display the carousel.

Set

data-ur-set="carousel"

Components

Required

  • data-ur-carousel-component="view_container" - this div should wrap the whole carousel; it must be set to no overflow.
  • data-ur-carousel-component="scroll_container" - this div should wrap the scrolling part of the carousel
  • data-ur-carousel-component="item" - this attribute is for each item that needs to be displayed in the carousel; must have float:left and display:inline styles

Optional

  • data-ur-carousel-component="button" - indicates the previous and next buttons
  • data-ur-carousel-button-type="prev" - specifies the "previous" button
  • data-ur-carousel-button-type="next" - specifies the "next" button
  • data-ur-carousel-component="count" - gives an indicator of the position of the carousel, in the form "1 of 8", etc.
  • data-ur-touch="disabled" - by default, this is set to enabled; setting it as disabled stops the touch function

div[data-ur-carousel-component='view_container'] {
  height: 270px;
  width: 100%;
  overflow-x: hidden; }
div[data-ur-carousel-component="scroll_container"] {
  display: block; }
div[data-ur-carousel-component="scroll_container"] > * {
  display: inline-block;
  float: left; }
div[data-ur-carousel-component="button"][data-ur-state="disabled"] {
  opacity: 0.3; }
