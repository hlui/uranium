Carousels

Description

The carousel is a widget to allow for horizontally scrolling (with touch or buttons) between a set of items.

Components

Required

Firstly, we wrap the whole widget in a tag with the attribute data-ur-set="carousel".

The main attribute we need to use for the carousel is:

data-ur-carousel-component

To this attribute, we add values described below,

  • "view_container"
         - for the div that wraps the whole carousel. It has no overflow, so you only see one item at a time.
  • "scroll_container"
         - for the div that wraps the scrolling bit of the carousel
  • "item"
         - for each item to be displayed in the carousel. They can be <div> or <img> tags. In the CSS, each item needs to have float:left and display:inline stylings applied.

We need to add a special attribute to the previous and next buttons. The previous button needs the attribute data-ur-carousel-button-type="prev", and the next button needs (correspondingly) data-ur-carousel-button-type="next".

Optional

  • "count"
         - this gives us an indicator of the position of the carousel, in the form '1 of 8', etc.
  • "button"
         - for the previous and next buttons.
